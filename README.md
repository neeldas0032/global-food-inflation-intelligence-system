# 🌍 Global Food Inflation Intelligence System

## 📊 Project Overview

The **Global Food Inflation Intelligence System** is an end-to-end data analytics project designed to monitor, analyze, and visualize global food price trends. Using large-scale datasets from the World Food Programme, the system identifies food inflation patterns, commodity price volatility, and high-risk regions experiencing food price instability.

This project combines **data engineering, exploratory data analysis, and business intelligence dashboards** to transform raw global food price data into actionable insights.

---

## 🎯 Project Objectives

* Analyze **global food price inflation trends** across countries.
* Identify **most expensive commodities and volatile markets**.
* Detect **regions with high food price instability**.
* Build an **interactive Power BI dashboard** for decision-makers.
* Generate insights that can support **policy planning and humanitarian response**.

---

## 🗂 Dataset

The dataset used in this project comes from the **World Food Programme Global Food Prices Database**, containing millions of food price records across multiple countries and commodities.

**Dataset Source:**
https://data.humdata.org/dataset/wfp-food-prices

The dataset includes the following attributes:

* Country code
* Administrative regions
* Market location
* Commodity category
* Commodity name
* Price
* Currency
* Date
* Latitude and longitude

## 📈 Key Analytics Performed

### 1️⃣ Global Food Inflation Index

Calculated a global index representing average food price changes over time to understand macro-level inflation trends.

### 2️⃣ Commodity Price Volatility Analysis

Measured volatility using statistical metrics to identify commodities with unstable pricing patterns.

### 3️⃣ Most Expensive Commodities

Identified commodities contributing the highest price levels across markets.

### 4️⃣ Countries with Highest Food Prices

Analyzed markets to detect countries with the highest aggregated food prices.

### 5️⃣ Category-Level Price Distribution

Visualized how different food categories contribute to global food pricing patterns.

### 6️⃣ Geospatial Food Price Mapping

Used geographic coordinates to create a **global food price map** for visual exploration.

## 📌 Key Insights
